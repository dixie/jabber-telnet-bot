jabber-telnet-bot by dixie

Jabber Bot which provides interface to telnet connection (over 23).

Jabber Bot which provides interface to telnet connection (over 23). Besically it allows interaction with some MUDs or Talkers over the IM clients (pidgin, gajim, gtalk, etc). FAQ: * Q: Is this transport for XMPP server which allows interactions with other XMPP servers and user can add Atlantis user into Contact List ? * A: No. This is bot which is something like a gateway over which user can create telnet session and interact with Atlantis or other telnet based server.

Dependencies

You need to have at least: * Python 2.5.2 * Python xmpp 0.3.1

Install

For debian-based distros (Ubuntu) type following: sudo apt-get install python-xmpp python And extract archive (where is also this README.txt placed) to some directory, e.g /home/yourname/jabberAtlBot/ or /usr/local/jabberAtlBot/ for system-wide
